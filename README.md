## Healthcare NLP Analysis

## Overview

This project analyzes patient feedback using Natural Language Processing (NLP) to extract meaningful insights and improve healthcare services.

---

## Objectives

* Perform sentiment analysis on patient reviews
* Identify key issues and positive feedback
* Visualize insights using Power BI

---

##  Tools Used

* Python (pandas, nltk, TextBlob)
* Power BI

---

##  Dashboard Features

* Sentiment Distribution (Bar Chart)
* Sentiment Percentage (Pie Chart)
* Word Cloud (Common Terms)
* KPI Cards (Total, Positive, Negative Reviews)

---

## Key Insights

* Majority feedback is positive
* Common complaints: waiting time, hygiene
* Positive mentions: doctor behavior, staff support
